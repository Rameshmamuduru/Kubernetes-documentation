```
| Controller                             | Purpose                                                                                         | Production Use Case                                                                                      |
| -------------------------------------- | ----------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------- |
| **Deployment**                         | Ensures desired number of **stateless Pod replicas**, handles **rolling updates and rollbacks** | Web apps, APIs, microservices                                                                            |
| **StatefulSet**                        | Manages **stateful Pods**, provides **stable network IDs and persistent storage**               | Databases (MySQL, MongoDB), Kafka, Cassandra                                                             |
| **DaemonSet**                          | Ensures **one Pod runs on each node** (or a subset of nodes)                                    | Logging agents (Fluentd), Monitoring agents (Prometheus Node Exporter), Network plugins (Calico, Cilium) |
| **ReplicaSet**                         | Ensures a fixed number of **Pod replicas**                                                      | Mostly managed by Deployment, rarely used directly                                                       |
| **ReplicationController (RC)**         | Old version of ReplicaSet                                                                       | Legacy apps; mostly deprecated                                                                           |
| **Job**                                | Runs **Pods to completion**, ensures a certain number of successful completions                 | Batch jobs, DB backups, data migration                                                                   |
| **CronJob**                            | Runs Jobs on a **schedule** (like cron)                                                         | Nightly backups, scheduled reports, periodic cleanup tasks                                               |
| **HorizontalPodAutoscaler (HPA)**      | Automatically scales Pods based on metrics (CPU, memory, custom)                                | Web apps, APIs under variable load                                                                       |
| **VerticalPodAutoscaler (VPA)**        | Adjusts Pod **resource requests (CPU/memory)** automatically                                    | Resource optimization for stable workloads                                                               |
| **Namespace controller**               | Ensures default objects exist in a Namespace, manages lifecycle                                 | Cluster management, namespace automation                                                                 |
| **ServiceAccount controller**          | Manages creation and lifecycle of ServiceAccounts                                               | Default accounts for Pods, RBAC management                                                               |
| **Endpoint controller**                | Maintains **EndPoints for Services**, keeps track of Pod IPs                                    | Service-to-Pod communication inside cluster                                                              |
| **PersistentVolume controller**        | Manages **PV binding** and dynamic provisioning                                                 | Automatically creates PVs when PVC requests are made                                                     |
| **PersistentVolumeClaim binder**       | Binds PVCs to suitable PVs                                                                      | Ensures applications get requested storage                                                               |
| **Node controller**                    | Monitors node health, evicts Pods from dead nodes                                               | High-availability production clusters                                                                    |
| **Job controller**                     | Monitors batch jobs, retries failed Pods                                                        | Batch workloads, migrations, cleanup scripts                                                             |
| **CronJob controller**                 | Schedules Jobs at defined intervals                                                             | Scheduled maintenance tasks                                                                              |
| **ResourceQuota controller**           | Enforces **resource quotas per namespace**                                                      | Prevents teams from overusing cluster resources                                                          |
| **LimitRange controller**              | Enforces **Pod/container resource limits**                                                      | Ensures fair resource allocation in namespaces                                                           |
| **Service controller**                 | Allocates **external IPs** for LoadBalancer services                                            | Exposing cloud apps to the Internet                                                                      |
| **EndpointSlice controller**           | Manages **EndpointSlices** for scalable networking                                              | Large-scale service discovery and routing                                                                |
| **GC (Garbage Collection) controller** | Cleans up unused resources (Pods, Jobs, etc.)                                                   | Keeps cluster clean, prevents orphan resources                                                           |
| **Namespace lifecycle controller**     | Manages creation and deletion of namespaces and associated objects                              | Multi-tenant cluster management                                                                          |

```
